# Sorting Algorithm Performance Analyzer

A simple Python program that compares different sorting algorithms using execution time, comparisons, swaps/writes, and space usage.

## Algorithms Included

- Selection Sort
- Bubble Sort
- Insertion Sort
- Merge Sort

## Test Cases

The program tests each algorithm on:

- 5 sorted values
- 5 reverse values
- 100 sorted values
- 100 reverse values

## How to Run

```bash
python sorting_performance_analyzer.py
